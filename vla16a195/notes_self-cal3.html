Self-Calibration III

2016-08-10

In the last month, several trials to improve the self-cal was done:

  • Initially clean conservatively (less iteration) then self-cal with less points in the model.

  • Use only several central pointings as model to self-cal.

Both of the two ideas (with various parameters combination) didn't give much improvements. The SNRs can be calculated as:

C-Band before self-cal:

imin = '7.4.0.image'

noise=imstat(imagename=imin,)['rms'][0]

peak=imstat(imagename=imin,)['max'][0]

print 'rms %8.4f, peak %8.3f, snr %5i' % (noise, peak, peak/noise) 

rms   0.0086, peak    1.051, snr   121

C-band bfter self-cal:

imin = '7.4.2.image'

noise=imstat(imagename=imin,)['rms'][0]

peak=imstat(imagename=imin,)['max'][0]

print 'rms %8.4f, peak %8.3f, snr %5i' % (noise, peak, peak/noise) 

rms   0.0086, peak    1.063, snr   122

X-Band before self-cal:

imin = '8.4.0.image'

noise=imstat(imagename=imin,)['rms'][0]

peak=imstat(imagename=imin,)['max'][0]

print 'rms %8.4f, peak %8.3f, snr %5i' % (noise, peak, peak/noise) 

rms   0.0027, peak    0.533, snr   195

X-band bfter self-cal:

imin = '8.4.2.image'

noise=imstat(imagename=imin,)['rms'][0]

peak=imstat(imagename=imin,)['max'][0]

print 'rms %8.4f, peak %8.3f, snr %5i' % (noise, peak, peak/noise) 

rms   0.0028, peak    0.549, snr   198

Almost no improvement.

The best images (and corresponding residual maps) as far as we can get now are:
