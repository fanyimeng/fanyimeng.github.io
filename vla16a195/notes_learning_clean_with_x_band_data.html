Case 1.4:

now try multiscale and also have nterms .

A good rule of thumb when using multiscale is 0, 2xbeam, 5xbeam (where beam is the synthesized beam), and larger scales up to about half the minor axis maximum scale of the mapped structure. (casaguide)

def copyImage(prefix, newprefix):
  suffix = ['.flux',
            '.flux.pbcoverage',
            '.image',
            '.model',
            '.psf',
            '.residual',]
  for i in range(len(suffix)):
    sysCommand = 'cp -r ./' + prefix + suffix[i] + "  ./" + newprefix + suffix[i]
    print sysCommand
    os.system(sysCommand)
  return 0
